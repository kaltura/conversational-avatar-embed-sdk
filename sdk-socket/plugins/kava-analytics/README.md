# Kaltura Avatar Analytics (KAVA Plugin)

Optional analytics plugin for the [Kaltura Avatar SDK (Socket)](../../README.md). Reports Immersive Agent lifecycle events and standard KAVA events to Kaltura's analytics endpoint.

## Installation

### CDN (recommended)

```html
<!-- After the core SDK -->
<script src="https://cdn.jsdelivr.net/gh/kaltura/conversational-avatar-embed-sdk@latest/sdk-socket/plugins/kava-analytics/kaltura-avatar-analytics.js"></script>
```

### Local

```html
<script src="plugins/kava-analytics/kaltura-avatar-analytics.js"></script>
```

## Quick Start

```javascript
const sdk = new KalturaAvatarSDK({ clientId: '...', flowId: '...', container: '#avatar' });

const kava = new KalturaAvatarAnalytics(sdk, {
  ks: 'your-kaltura-session-token',   // Required
  partnerId: 12345,                    // Required
  hostingApp: 'my-app',
  hostingAppVer: '1.0.0'
});

sdk.connect();
// Plugin auto-reports: callStarted, callEnded, messageResponse
```

## What's Tracked Automatically

| SDK Event | KAVA Event | Type |
|-----------|-----------|------|
| `ready` | callStarted | 80002 |
| `disconnected` / unload | callEnded (with duration) | 80003 |
| `avatar-speech` | messageResponse (experience=Call) | 80001 |
| `user-speech` (final only) | messageResponse (experience=Chat) | 80001 |
| `reconnected` | callEnded + new callStarted | 80003 + 80002 |

## Configuration

```javascript
new KalturaAvatarAnalytics(sdk, {
  // Required
  ks: string,           // Kaltura Session token (app-provided, never generated by plugin)
  partnerId: number,    // Kaltura partner ID

  // Optional (defaults shown)
  agentId: '',          // Auto-read from sdk.getFlowId()
  genieId: '',          // Auto-read from sdk.getClientId()
  clientTag: '',        // e.g., 'my-app:1.0.0'
  hostingApp: '',       // Application name for server-side enrichment
  hostingAppVer: '',    // Application version
  serviceUrl: 'https://analytics.kaltura.com/api_v3/index.php',
  requestMethod: 'POST',
  userId: '',           // User ID for attribution

  // Behavior
  autoStart: true,      // callStarted on 'ready'
  autoEnd: true,        // callEnded on disconnect/unload
  autoMessages: true,   // messageResponse on speech events
  beaconOnUnload: true, // sendBeacon for page-leave events

  // Hooks
  tamperHandler: null,  // fn(params) => boolean — return false to suppress

  debug: false
});
```

## API

### Manual Events

```javascript
kava.sendFeedback(messageId, 'like' | 'dislike');  // 80005
kava.pageView(pageName, pageType);                  // 10003
kava.buttonClick(name, value, context);             // 10002
kava.customEvent(eventType, { ...extraFields });    // any type
```

### Context (affects all subsequent events)

```javascript
kava.setContextId('3:earnings');     // e.g., slideNum:category
kava.setEntryId('entry_abc');
kava.setContextType(1);              // 1=Entry, 2=Channel, 3=Global
kava.setMetadata('customVar1', 'value');
kava.setKS('new-ks-token');          // Update KS mid-session
```

### Observability

```javascript
kava.on('sent', ({ eventType, params }) => { ... });
kava.on('error', ({ eventType, error }) => { ... });
kava.getStats();  // { eventsSent, errors, messageCount, callDuration, threadId, sessionId }
```

### Lifecycle

```javascript
kava.destroy();  // Fires callEnded if active, unsubscribes all handlers
```

## Static Constants

```javascript
KalturaAvatarAnalytics.EventType    // { MESSAGE_RESPONSE, CALL_STARTED, CALL_ENDED, ... }
KalturaAvatarAnalytics.ExperienceType  // { CHAT: 1, CALL: 2 }
KalturaAvatarAnalytics.ResponseType    // { TEXT: 1, FLASHCARD: 2 }
KalturaAvatarAnalytics.ReactionType    // { LIKE: 1, DISLIKE: 2 }
KalturaAvatarAnalytics.ContextType     // { ENTRY: 1, CHANNEL: 2, GLOBAL: 3 }
```

## KS Responsibility

The app provides the KS — this plugin never generates, refreshes, or validates it. If the app refreshes the KS mid-session, call `kava.setKS(newKS)`.

**Recommended:** Use a Widget Session KS from `session/action/startWidgetSession` with widget ID `_<partnerId>`. See **[KS_GUIDE.md](./KS_GUIDE.md)** for complete generation instructions, security notes, and privilege embedding.

## Transport

- **Normal events**: `fetch()` with `keepalive: true` (HTTP POST)
- **Page-leave events**: `navigator.sendBeacon()` (fired on `visibilitychange: hidden`)
- **No retry/batching** — fire immediately per event
